# HamsterTodo — Product Requirements Document

## Problem

개발자/지식근로자는 Obsidian 등 노트앱에 할 일을 적어두지만, 아침에 노트를 다시 열어보지 않으면 잊어버린다. 할 일 관리 앱은 이미 넘쳐나지만, 기존 워크플로우(Obsidian 일일노트)를 버리고 새 앱에 다시 입력하는 건 마찰이 크다.

**핵심 인사이트**: 할 일을 "또 입력"하는 게 아니라, 이미 적어둔 할 일을 "알려주기만" 하면 된다.

## Target User

- Obsidian으로 일일노트를 쓰는 개발자/지식근로자
- macOS 사용자
- 할 일 관리에 별도 앱을 추가하고 싶지 않은 미니멀리스트

## Value Proposition

| 기존 대안 | 문제 | HamsterTodo |
|-----------|------|-------------|
| Todoist, Things 등 | 이중 입력 필요, 노트와 분리됨 | Obsidian 노트를 그대로 읽음 — 추가 입력 제로 |
| Obsidian 직접 열기 | 아침에 안 열면 끝 | 메뉴바 상주 + 9시 알림으로 자동 리마인드 |
| macOS 미리알림 | Obsidian 연동 없음 | 마크다운 파싱으로 네이티브 연동 |

**한 줄 요약**: Obsidian 일일노트를 읽어서, 메뉴바 햄스터가 매일 아침 할 일을 알려주는 macOS 앱.

## Core Features (MVP)

| 우선순위 | 기능 | 상태 |
|---------|------|------|
| P0 | Obsidian 일일노트 파싱 (`## 내일 할 일` 섹션) | Done |
| P0 | 메뉴바 상주 픽셀 햄스터 (3프레임 애니메이션) | Done |
| P0 | 클릭 시 할 일 목록 팝업 | Done |
| P0 | 매일 9시 macOS 알림 | Done |
| P1 | Obsidian 볼트 폴더 선택 UI (Sandbox 대응) | Done |
| P1 | 로그인 시 자동 실행 | Done |

## User Scenarios

### 시나리오 1: 첫 설정
1. 앱 설치 후 실행 — 메뉴바에 햄스터 등장
2. 햄스터 클릭 → "폴더 선택..." 버튼 표시
3. Obsidian 일일노트 폴더(`01-daily/`) 선택
4. 즉시 할 일 목록 표시 — 설정 끝

### 시나리오 2: 매일 아침 루틴
1. 9시에 macOS 알림: "쪽쪽! 오늘 할 일을 가져왔어요~"
2. 알림 클릭 또는 메뉴바 햄스터 클릭
3. 팝업에서 오늘 할 일 확인
4. 팝업 외부 클릭 → 닫힘, 업무 시작

### 시나리오 3: 중간 점검
1. 오후에 "뭐 남았더라?" → 메뉴바 햄스터 클릭
2. 완료/미완료 항목 한눈에 확인
3. Obsidian에서 `- [x]`로 체크하면 다음 클릭 시 반영

## Success Metrics

| 지표 | 목표 | 측정 방법 |
|------|------|----------|
| Daily Active Usage | 앱 설치 후 7일 연속 사용 | 추후 opt-in 텔레메트리 |
| Setup Completion | 첫 실행 → 폴더 선택 완료 90%+ | 앱 내 상태 |
| Retention | 30일 후에도 로그인 항목에 유지 | 추후 측정 |

## Non-goals (의도적으로 하지 않는 것)

- **할 일 편집**: 앱에서 마크다운 파일을 수정하지 않는다. 편집은 Obsidian에서.
- **할 일 앱 되기**: 캘린더, 반복 태스크, 프로젝트 관리 등 추가하지 않는다.
- **크로스 플랫폼**: macOS 전용. iOS/Windows 지원하지 않는다.
- **Obsidian 플러그인**: 독립 앱으로 유지. Obsidian API에 의존하지 않는다.
- **네트워크 통신**: 로컬 파일만 읽는다. 서버, 동기화, 계정 없음.

## Future Considerations

다음은 MVP 이후 사용자 피드백에 따라 검토할 항목이다. 지금 구현하지 않는다.

- **커스텀 섹션 헤딩**: `## 내일 할 일` 외 다른 헤딩 지원
- **알림 시간 설정**: 9시 고정이 아닌 사용자 지정
- **햄스터 스킨/테마**: 다양한 캐릭터 또는 컬러 팔레트
- **여러 볼트 지원**: 복수 Obsidian 볼트 동시 모니터링
- **파일 변경 감지**: 폴더 워치로 실시간 업데이트 (현재는 클릭 시 로드)
- **App Store 배포**
